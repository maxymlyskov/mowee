2 СТРАНИЦА

Объект View для набора текста будет находится сверху на растоянии примерно 15% от начала экрана;
После набора текста в поиск и нажатия кнопки поиска берутся данные с АПИ и высвечиваются блоки объекта View с подходящими ответами;
Справа сверху блока будет кнопка лайк при нажатии на которую фильм добавляется в библиотеку отдельно взятого пользователя;
80% отдельного блока будет состоять из картинки фильма, 20% - текст;
Текст будет включать название фильма (возможно год показа);
При клике на блок будет предоставляться детальная информация о фильме (режиссер, актеры, жанр, короткое описание, рейтинг);
С помощью объекта Modal с окна, что показывает детали фильма свайпом можно будет выйти назад;
